# Eminence Guidance Prediction

## Overview

This project aims to develop a machine learning based approach to estimate articular eminence inclination from routinely available radiographic measurements. The larger clinical goal is to explore whether these predicted values can support more individualized sagittal articulator programming in prosthodontic practice.

In conventional prosthodontic workflows, sagittal condylar guidance is often set using average values or direct clinical records. While direct records can improve accuracy, they may not always be practical in every setting. This project investigates whether cephalometric and panoramic radiographic parameters can be used to predict radiographic articular eminence inclination, which may then serve as a reference for articulator programming.

## Problem Statement

Articular eminence inclination influences mandibular movement and plays an important role in occlusal rehabilitation and articulator adjustment. However, individualized programming is not always performed because of time constraints, lack of records, or limited access to advanced methods.

The problem we are trying to solve is:

Can we build a predictive model that uses common radiographic measurements to estimate articular eminence inclination for a given patient?

If successful, such a model could provide a simple and accessible way to support more patient specific articulator settings.

## Objective

The objective of this project is to train and evaluate regression based machine learning models that predict mean articular eminence inclination using selected cephalometric and panoramic parameters.

## Input Variables

The model uses radiographic features such as:

SNA  
SNB  
ANB  
SN GoGn  
Occlusal plane angle  
Ramus height  
Condylar height  
Bigonial width  
Condylar asymmetry index  

## Target Variable

The target variable is:

Mean articular eminence inclination

This is computed as the average of the right and left articular eminence inclination values.

## Clinical Relevance

This work is relevant to prosthodontics because it explores a practical method for approximating sagittal guidance values using radiographic data that may already be available as part of routine records.

The long term goal is not to replace clinical judgment or direct functional records, but to investigate whether predictive modeling can act as an adjunctive tool for individualized articulator programming, especially in full mouth rehabilitation and other complex restorative cases.

## Current Scope

This repository currently focuses on:

Data preparation  
Feature selection  
Model development  
Model evaluation  
Comparison of predicted values with collected patient values  

## Future Scope

## Repository Goal

The goal of this repository is to provide a reproducible workflow for studying the relationship between craniofacial radiographic parameters and articular eminence inclination using machine learning.

## Disclaimer

This repository is intended for academic and research purposes. The predictions generated by these models should not be used as a direct substitute for comprehensive clinical evaluation.
